# Student Management API

A RESTful API built using Spring Boot and MySQL for managing students.

## Features

- Create student
- Get all students
- Get student by ID
- Update student
- Delete student

## Technologies

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- ## API Endpoints

POST /api/students

GET /api/students

GET /api/students/{id}

PUT /api/students/{id}

DELETE /api/students/{id}
